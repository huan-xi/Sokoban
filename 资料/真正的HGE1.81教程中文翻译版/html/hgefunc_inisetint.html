HGE::Ini_SetInt

将一个整型值写入初始化文件。 

void Ini_SetInt(
  const char *section,
  const char *name,
  int value
);

参数

section
初始化文件的段落(Section)。
name
键名。
value
要写入的值。

备注

调用此函数前需要先设置系统标记HGE_INIFILE。

如果初始化文件，段落(Section)或键不存在则创建他们。段落(Section)名不区分大小写(case-independent也不知是区分还是不区分了迷糊了)。

需要

头文件: hge.h
库文件: hge.lib
