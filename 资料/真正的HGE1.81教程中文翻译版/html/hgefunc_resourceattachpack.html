HGE::Resource_AttachPack

附加(attach)一个资源文件。 Attaches a resource pack. 

bool Resource_AttachPack(
  const char *filename,
  const char *password = 0
);

参数

filename
资源包文件名(相对[相对于程序目录]路径或绝对路径).
password
可选的密码参数

返回值

成功返回true，失败则返回false。

备注

如果函数调用失败，你可以使用System_GetErrorMessage函数获得错误信息。 如果设置了系统状态HGE_LOGFILE，错误信息将被写入日志。

当加载资源时，越晚加载的包具有越高的优先级。 更多资源包相关的信息请参考Compiling resource pack部分。

当资源包不再有用时，必须使用Resource_RemovePack移除。 所有未被移除的包都会在调用Release函数时被删除。

需要

头文件: hge.h
库文件: hge.lib

另见

Resource_RemovePack, Resource_RemoveAllPacks, Resource_Load,
