hgeSprite::GetTextureRect

返回当前精灵使用的纹理区域。 

void GetTextureRect(
  float *x,
  float *y,
  float *w,
  float *h
);

参数

x
float型指针，用于存储当前纹理区域的左上角X坐标。
y
float型指针，用于存储当前纹理区域的左上角Y坐标。
w
float型指针，用于存储当前纹理区域的宽度。
h
float型指针，用于存储当前纹理区域的高度。

需要

头文件: hgesprite.h
库文件: hgehelp.lib

另见

SetTextureRect, SetTexture, GetTexture
