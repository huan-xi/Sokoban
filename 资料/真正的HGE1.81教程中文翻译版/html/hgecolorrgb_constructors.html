hgeColorRGB构造函数

要创建并初始化一个hgeColorRGB对象你可以使用如下构造函数之一： 

hgeColorRGB();

hgeColorRGB(
  DWORD color
);

hgeColorRGB(
  float r,
  float g,
  float b,
  float a
);

hgeColorRGB(
  const hgeColorRGB &col
);

参数

color
用于初始化hgeColorRGB对象的硬件颜色。
r
红色值范围从0.0到1.0。
g
绿色值范围从0.0到1.0。
b
蓝色值范围从0.0到1.0。
a
透明度值范围从0.0到1.0。

需要

头文件: hgecolor.h
