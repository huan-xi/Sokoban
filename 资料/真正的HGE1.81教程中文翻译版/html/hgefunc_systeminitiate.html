HGE::System_Initiate

初始化引擎所需的所有软硬件并创建程序窗口。 

bool System_Initiate();

返回值

如果成功则返回true。否则返回false

备注

在调用此函数之前可使用System_SetState设置所需的系统标记。 Before calling this function set desired system states with System_SetState calls. 虽然绝大多数的系统标记都有默认值并可以再之后修改，但是需要注意的是>HGE_FRAMEFUNC必须被设置。

Timer_*, Effect_*, Music_*, Stream_*, Channel_*, Input_*, Gfx_*, Target_* and Texture_* 函数群在系统初始化前无法使用。 其他HGE函数可以再任何位置使用。

如果函数调用失败，你可以通过System_GetErrorMessage获得错误信息。 If the function fails, you may obtain error message with System_GetErrorMessage function. 如果设置了系统标记HGE_LOGFILE，错误信息将被写入日志。

程序推出前必须调用System_Shutdown复原视频模式，并释放分配的资源。

需要

头文件: hge.h
库文件: hge.lib

另见

System_SetState, System_GetErrorMessage, System_Shutdown
