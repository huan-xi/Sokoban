HGE::Input_KeyUp

测试前一个帧函数调用过程中是否有按键被释放。 

bool Input_KeyUp(
  int key
);

参数

key
指定的虚拟键值。 更多信息请参考键值部分

返回值

如果前一个框架调用过程中有按键被释放，则返回true，否则返回false。

备注

Input_KeyUp处理键盘和鼠标按键。

按键状态在用户的帧函数外被更新，所以不要在循环中使用Input_KeyUp判断一个按键是否被按下或释放，这将造成系统挂起。

需要

库文件: hge.lib
