HGE::Ini_GetFloat

从初始化文件中读取一个浮点值。 

float Ini_GetFloat(
  const char *section,
  const char *name,
  float def_val
);

参数

section
初始化文件的段落(Section)名。
name
键名。
def_val
默认值。

返回值

返回从初始化文件读取的值。如果该文件，段落(Section)或键值不存在，则返回默认值。

备注

在调用此函数之前必须设置系统标记HGE_INIFILE的值。 段落(Section)名不区分大小写(case-independent)。

需要

头文件: hge.h
库文件: hge.lib
