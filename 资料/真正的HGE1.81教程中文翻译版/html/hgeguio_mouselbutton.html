hgeGUIObject::MouseLButton

通知控件鼠标左键状态改变。 

virtual bool MouseLButton(
  bool bDown
);

参数

bDown
如果鼠标左键被按下，bDown是true。 如果鼠标左键被释放，bDown是true。

返回值

如果控件已经编辑了它自身的状态，并且想要通知调用者， MouseLButton应返回true。否则它应返回false。

备注

hgeGUIObject有默认的实现， 因此如果没有必要，你可以不必重定义该方法。

需要

头文件: hgegui.h

另见

MouseRButton, MouseMove, MouseWheel
