HGE::Music_SetInstrVolume

设置一段音乐中指定乐器的音量。 

void Music_SetInstrVolume(
  HMUSIC music,
  int instr,
  int volume
);

参数

music
音乐句柄。
instr
受影响的乐器编号。
Number of instrument to affect.
volume
乐器的新的音量，0..100。

需要

头文件: hge.h
库文件: hge.lib

另见

Music_GetInstrVolume, Music_SetChannelVolume, Music_GetChannelVolume
