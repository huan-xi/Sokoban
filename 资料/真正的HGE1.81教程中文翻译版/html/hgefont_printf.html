hgeFont::printf

渲染格式化的文本字符串。 

void printf(
  float x,
  float y,
  int align,
  const char *format,
  ...
);

参数

x
屏幕X坐标。
y
屏幕Y坐标。
align
排列分类符。可以使如下之一: HGETEXT_LEFT, HGETEXT_RIGHT, HGETEXT_CENTER.
format
C语言printf样式格式化规范。
...
要被渲染到屏幕上的可选参数列表。

备注

格式化规范的详细信息见编译器在线帮助或C语言参考手册。

需要

头文件: hgefont.h
库文件: hgehelp.lib

另见

Render, printfb
