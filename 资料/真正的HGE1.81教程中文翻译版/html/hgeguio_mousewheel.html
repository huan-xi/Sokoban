hgeGUIObject::MouseWheel

通知控件鼠标滚轮滚动。 

virtual bool MouseWheel(
  int nNotches
);

参数

nNotches
鼠标滚轮滚动刻度单位数。 正值代表鼠标滚轮向前滚动，远离用户； 负值代表鼠标滚轮向后滚动，朝向用户。

返回值

如果控件已经编辑了它自身的状态，并且想要通知调用者， MouseWheel应返回true。否则它应返回false。

备注

hgeGUIObject有默认的实现， 因此如果没有必要，你可以不必重定义该方法。

需要
