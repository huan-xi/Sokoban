hgeGUI::Update

更新GUI控件并处理用户输入。 

int Update(
  float fDeltaTime
);

参数

fDeltaTime
自上一次调用Update方法至今的时间间隔，以秒为单位。

返回值

如果某个控件改变了状态，Update将返回控件的标识。如果调用了Leave方法 并且所有的控件都完成了结束动画，Update将返回-1。其他情况返回0。

备注

在你的帧函数中调用Update方法来更新GUI。 自上一次调用帧函数至今的时间量可以通过系统层函数Timer_GetDelta获得。

需要

头文件: hgegui.h
库文件: hgehelp.lib
