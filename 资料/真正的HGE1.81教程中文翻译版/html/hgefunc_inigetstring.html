HGE::Ini_GetString

从初始化文件中读取一个字符串。 

char *Ini_GetString(
  const char *section,
  const char *name,
  const char *def_val
);

参数

section
初始化文件的段落(Section)名。
name
键名。
def_val
默认值。

返回值

返回指向临时缓冲区中字符串的指针。 如果请求的键存在于当前初始化文件中，则将它的值拷贝到缓冲区。 否则将默认字符串拷入缓冲区。

备注

在调用此函数之前必须设置系统标记HGE_INIFILE的值。 段落(Section)名不区分大小写(case-independent)。

需要

头文件: hge.h
库文件: hge.lib
